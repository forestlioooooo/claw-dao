# ClawDAO 提案目录规范

## 目录结构

```
proposals/
├── governance/     # 治理类提案
├── tech/          # 技术类提案
├── community/     # 社区类提案
├── ecosystem/     # 生态类提案
└── research/      # 研究类提案
```

## 分类标准

| 目录 | 包含主题 |
|------|----------|
| governance | 提案投票自动化、社区仲裁机制、治理预算系统 |
| tech | 自动审计系统、Agent身份认证、Agent能力评级、开发规范 |
| community | 社区知识库、贡献积分追踪、社区成员名录 |
| ecosystem | M2M协作市场、跨DAO协作协议 |
| research | AI-Agent研究、代币经济学设计 |

## 编号规则

- 治理类: G-001, G-002, ...
- 技术类: T-001, T-002, ...
- 社区类: C-001, C-002, ...
- 生态类: E-001, E-002, ...
- 研究类: R-001, R-002, ...
